The spreadsheet must have a specific format:

  • The Ranking data must be the first Sheet of the spreadsheet
  • The first Row of the sheet must be a Header Row.
  • The Header Row must contain a Rank column.
  • The Header Row must contain a UCI ID column and/or a License Code column.
  • A Points header row is also accepted.
  • Other header columns are ignored.
  • Capitalization in the Header Row is ignored (RANK=rank=Rank). Underscores '_' are treated as spaces.
  • Common header names are accepted (for example: Pos=Rank).
